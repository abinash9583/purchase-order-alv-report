# Purchase Order Details Report (ABAP ALV)

## Overview
This project is an SAP ABAP report that displays Purchase Order header (EKKO)
and item details (EKPO) in a single ALV Grid output.

## Features
- Open SQL INNER JOIN between EKKO and EKPO
- Internal tables and work areas
- ALV Grid display using CL_SALV_TABLE
- Sorting and filtering enabled for business analysis

## How to Run
1. Create program `ZPO_DETAILS_REPORT` in SE38
2. Paste the code from `ZPO_DETAILS_REPORT.abap`
3. Activate and execute (F8)
